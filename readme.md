# Clockify

This is basically a simple timer app. user can start,pause and reset the timer as their need.

## Installation

Install my-project with npm

```bash
  git clone https://github.com/Arifuzzaman-Nishan/timer_app.git
  npm install timer_app
  cd timer_app
```
